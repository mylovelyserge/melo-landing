# Melo 🎵

> Describe your mood. Get the perfect playlist.

Melo is an AI-powered Chrome extension that finds 
music for your mood in seconds. Type or speak how 
you feel — Spotify does the rest.

## How it works
1. Click the Melo icon in your browser
2. Type or speak your mood
3. Get matched tracks — opens instantly in Spotify

## Stack
- Chrome Extension (Manifest V3)
- Web Speech API (voice input)
- Anthropic Claude API (mood analysis)
- Spotify Web API (track discovery)
